# NYC Yellow Taxi Data Pipeline

End-to-end data engineering project analyzing NYC taxi trip data with automated ELT, transformation, and visualization.

## Architecture
[Raw Data] → [PostgreSQL] → [dbt] → [Analytics Tables] → [Streamlit Dashboard]

## Tech Stack
- **Database**: PostgreSQL
- **Transformation**: dbt Core
- **Visualization**: Streamlit, Plotly
- **Languages**: Python, SQL
- **Containerization**: Docker(local database env)

## Key Features
- Automated data ingestion from NYC TLC public dataset
- Data quality checks and deduplication
- 5+ analytics models (daily summaries, hourly demand, payment analysis)
- Interactive dashboard with 10+ visualizations
